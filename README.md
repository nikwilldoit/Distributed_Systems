# Distributed_Systems
ΚΑΤΑΝΕΜΗΜΕΝΑ ΣΥΣΤΗΜΑΤΑ Εαρινό Εξάμηνο 2025-2026 Υποχρεωτική εργασία

## Σχετικά με την εργασία

Ο σχεδιασμός του κατανεμημένου συστήματος κατασκευάζεται υπόψη 2 χρηστών:

* Manager(Διαχειριστής): Υπεύθυνος για την διαχείριση και την επεξεργασία διαθέσιμων λειτουργιών του casino.
* Player(Παίκτης): Ο χρήστης που θα παίζει στην εφαρμογή με τα παιχνίδια που του είναι διαθέσιμα.

### Αρμοδιότητες Manager

Ο Manager Θα είναι σε θέση, από τα παιχνίδια που υπάρχουν στο σύστημα:

1. Να τα προσθέτει/αφαιρεί απο το casino, όποτε επιθυμεί.
2. Να τροποποιεί ρυθμίσεις των διαθέσιμων παιχνιδιών.
3. Να εμφανίζει κέρδοι και απώλειες ανα παίκτη/παιχνίδι.

Το παραπάνω σύστημα θα υλοποιείται με cli. Ο Manager θα συνδέεται στο server και μέσο του τερματικού, θα είναι σε θέση να ρυθμίζουν το σύστημα και να αλλάζουν τις ρυθμίσεις του.

### Δραστηριότητες Χρήστη

Η διεπαφή χρήστη θα υλοποιηθεί με android studio σε Java.Ο χρήστης θα πρέπει να μπορεί:

1. Να βλέπει τα διαθέσιμα παιχνίδια.
2. Να μπορεί να επιλέξει να παίξει από τα διαθέσιμα παιχνίδια.
3. Να ανάζητηση παιχνίδια με βάση κάποιο κριτήριο.
4. Να μπορεί να βαθμολογήσει τα παιχνίδια.
